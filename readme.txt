This plugin will remove script and version numbers from the source of your pages.

== Description ==
Blockinator lets you disable the XMLRPC script.

You can use blockinator to remove these scripts from your source code of Wordpress:
- xmlrpc
- version numbers from script url
- shortlinks
- links to feeds
- dns prefetch option

== Installation ==
= From your WordPress Dashboard =

1. Visit “Plugins” → Add New
2. Search for Blockinator
3. Activate Blockinator from your Plugins page.

= From WordPress.org =

1. Download Blockinator
2. Upload the “blockinator” directory to your “/wp-content/plugins/” directory, using ftp, sftp, scp etc.
3. Activate Blockinator from your Plugins page.
